# 🌐 Proyecto 2: Riesgos Laborales en Informática 💻⚠️

Este repositorio contiene el trabajo colaborativo sobre **riesgos laborales y prevención en el ámbito informático**.  
El objetivo principal es **analizar los riesgos al trabajar con equipos informáticos** y **proponer medidas de prevención y buenas prácticas**.

---

Puedes ver el resultado final del proyecto aquí:  
👉 [Ir a la web](https://juliogutierrezfi.github.io/seguridad-trabajo-inform-tica/)

---

## 📚 Contenidos del Proyecto

La web aborda los siguientes puntos clave:

1. **¿Qué son los riesgos laborales?**  
   Introducción a los riesgos presentes en el entorno de trabajo informático.

2. **Riesgos físicos en el uso de ordenadores**  
   Problemas de postura, fatiga visual, lesiones musculares y otros riesgos ergonómicos.

3. **Riesgos eléctricos y de equipos informáticos**  
   Medidas de seguridad al utilizar dispositivos electrónicos y prevenir accidentes.

4. **Ciberseguridad y riesgos digitales**  
   Protección de datos, contraseñas seguras y prevención de ataques informáticos.

5. **Medidas de prevención**  
   Buenas prácticas para mantener un entorno de trabajo seguro y saludable.
