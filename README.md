# Proyecto 2: Riesgos Laborales en Informática 💻⚠️
Este repositorio contiene el trabajo colaborativo sobre **Riesgos laborales y prevención en el ámbito informático**.  
El objetivo es analizar los posibles riesgos que existen al trabajar con equipos informáticos y proponer medidas para prevenirlos.

## 🚀 Ver la página web
Puedes ver el resultado final del proyecto aquí:
👉 **[https://JoseManuelPerez10.github.io/seguridad-trabajo-informática/]**

## 📚 Contenidos del Proyecto
La página aborda los siguientes puntos clave:

1. **¿Qué son los riesgos laborales?**  
   Introducción a los riesgos en el entorno de trabajo.

2. **Riesgos físicos en el trabajo con ordenadores**  
   Problemas de postura, fatiga visual y lesiones musculares.

3. **Riesgos eléctricos y de equipos informáticos**  
   Uso seguro de dispositivos electrónicos y prevención de accidentes.

4. **Ciberseguridad y riesgos digitales**  
   Protección de datos y seguridad en el entorno informático.

5. **Medidas de prevención**  
   Buenas prácticas para trabajar de forma segura con tecnología.
